# Dice Simulator (Python)

## Description
This project is a simple Dice Simulator built using Python. It simulates the rolling of a dice by generating a random number between 1 and 6 each time the user chooses to roll. The user can continue rolling the dice or exit the program anytime.

## Features
- Random dice roll between 1 and 6
- User can roll multiple times
- Exit option available
- Simple command-line interface

## Technologies Used
- Python
- random module
